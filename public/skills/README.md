# Skill icons

Place your skill images here. The app expects these filenames (lowercase):

**Programming:** `java.png`, `python.png`, `javascript.png`  
**Web:** `react.png`, `nodejs.png`, `express.png`, `html5.png`, `css3.png`  
**Database:** `mongodb.png`, `mysql.png`  
**DevOps:** `git.png`, `github.png`  
**UI/UX:** `figma.png`, `canva.png`, `drawio.png`, `lucidchart.png`  
**Others:** `oop.png`, `dsa.png`  
**Soft skills:** `problem-solving.png`, `time-management.png`, `communication.png`, `teamwork.png`

If an image is missing or fails to load, the skill box will show the abbreviation (e.g. JA, PY) instead.
